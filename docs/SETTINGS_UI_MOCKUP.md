## Color Scheme

### Light Mode
- Background: `NSColor.controlBackgroundColor` (Light Gray)
- Text: `NSColor.labelColor` (Black)
- Accent: `NSColor.accentColor` (Blue)
- GroupBox: `NSColor.windowBackgroundColor` (White)
- Secondary Text: `NSColor.secondaryLabelColor` (Gray)

### Dark Mode
- Background: `NSColor.controlBackgroundColor` (Dark Gray)
- Text: `NSColor.labelColor` (White)
- Accent: `NSColor.accentColor` (Blue)
- GroupBox: `NSColor.windowBackgroundColor` (Darker Gray)
- Secondary Text: `NSColor.secondaryLabelColor` (Light Gray)

- **Style:** `.switch`

- **Style:** `.radioGroup`
- **Spacing:** 8px vertical
- **Font:** System 13pt

### List (Per-App)
- **Height:** 200px fixed
- **Row Height:** Auto (based on content)
- **Separator:** Default system divider
- **Background:** Clear

### Buttons
- **Style:** `.plain` (text buttons)
- **Destructive:** Red foreground color
- **Icon Buttons:** SF Symbols 16pt

## Responsive Behavior

### Window Resizing
- **Fixed Size:** 600x500 (non-resizable)
- **macOS Standard:** Settings windows typically have fixed size
- **Content:** Scrollable vertically within fixed frame

### Tab Content
- Each tab uses `ScrollView` wrapper
- Content padding: 20px (applied to TabView)
- Spacing between sections: 20px
- TabView handles padding automatically

### Per-App List
- Fixed height (200px)
- Internal scrolling when > 5 apps
- Horizontal layout fills available width

### VoiceOver Labels
- All toggles have descriptive labels
- Buttons announce their action
- Tab buttons announce selected state
- List items read app name and state

### Keyboard Navigation
- Tab key cycles through controls
- Arrow keys navigate between tabs
- Space/Enter activates buttons/toggles
- Escape closes window

### Focus Indicators
- System default focus ring
- High contrast mode supported
- Reduced motion supported (no animations)
