## What is Text Expansion?

Text Expansion allows you to create shortcuts that automatically expand into longer text.

### Examples:
- Type `brb` ‚Üí expands to `be right back`
- Type `addr` ‚Üí expands to `123 Main Street, Hanoi, Vietnam`
- Type `xc` ‚Üí expands to `xin ch√†o`

---

## Getting Started

1. Open GoxViet Settings
2. Navigate to **Text Expansion** tab
3. Toggle **"Enable Text Expansion"** ON

---

## Creating Shortcuts

### Add Shortcut
1. Click **"Add Shortcut"** button
2. Enter **Trigger** (max 20 chars, alphanumeric only)
3. Enter **Replacement** (max 1000 chars)
4. Click **"Save"**

### Validation
‚úÖ Valid: `brb`, `addr`, `fn`
‚ùå Invalid: `b r b` (spaces), `@email` (special chars)

## Managing Shortcuts

- **Edit**: Click shortcut row
- **Delete**: Swipe left ‚Üí Delete
- **Enable/Disable**: Click checkbox

## Import & Export

- **Export**: Save all shortcuts as JSON

## Troubleshooting

### Shortcut Not Expanding
1. Check "Enable Text Expansion" is ON
2. Verify shortcut enabled (checkbox)
3. Restart GoxViet
